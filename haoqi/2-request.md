### 项目概况

名称：好奇猫迷你版

用户前端域名：tiger.haoqicat.com

管理员界面域名：ad.tiger.haoqicat.com

API 域名：tiger.haoduoshipin.com


### 用户前端首页

- 课程分类标签，例如： Linux Git React API ...
- 每个课程标签点开，都能显示这个标签下的相应几门课程
- 首页给出最新三门课程，已经指向所有课程页面的链接


### 管理员界面

- 管理员可以对标签进行增删
- 管理员可以对课程进行增删

### 响应式要求

手机和台式电脑即可。
